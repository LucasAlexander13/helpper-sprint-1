# Exercício 3

## Descrição da Sprint

3 - Crie uma função que receba uma lista (valide se é númerica) e retorne outra contendo:
- a) O maior valor;
- b) O menor valor;
- c) O valor médio.

Demonstre essa função com o paradigma funcional e imperativo.
